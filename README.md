# Projeto calculadora 
## Python e shell script

### Repositório com calculadora simples executável em python !
* Adição
* Subtração
* Divisão
* Multiplicação

### Arquivo de execução :
* calculadora.py

### Shell Script :
#### script para executar a calculadora
* chmod +x python_exe_script.sh
#### tornando o script executável :
* ./python_exe_script.sh
#### permissões 
* chmod 744 python_exe_script.sh

### Requisitos:
* Python 3
* Terminal/Linux/wsl/Ubuntu

Ajudar novos desenvolvedores a criarem seus primeiros projetos !
